## Features

- **Cross-Platform**: Built with Flutter for seamless performance on both mobile and desktop
- **Live TV Streaming**: Watch your favorite TV channels with support for M3U8 playlists
- **Movie Library**: Browse and stream on-demand movies with an intuitive interface
- **Local Storage**: SQL and NoSQL options for storing favorites and watch history
- **User Profiles**: Multiple user support with personalized recommendations

## Technical Details

- **Frontend**: Flutter with Material Design
- **State Management**: Provider pattern for efficient state management
- **Database**: Local SQLite and NoSQL solutions for offline functionality
- **Video Playback**: Custom video player with HLS/HTTP streaming support
- **Responsive Design**: Adapts to different screen sizes and orientations

## Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Android Studio / Xcode (for mobile development)
- VS Code (recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dordazrodriguez/IPTV-and-Movie-App.git
   cd IPTV-and-Movie-App
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```
